Ahorcado en c++

Paso a paso

Introducción

¿Qué es C++?

C++ es un lenguaje de programación que proviene de la extensión del lenguaje C para que pudiese manipular objetos.

¿Qué es el juego del ahorcado?

El ahorcado es un juego de dos o varios jugadores, el cual consite en que un jugador piensa una palabra y los demás deben intentar adivinarla.

Pasos

1Primer paso

Se agregan las bibliotecas iostream y string iostream es para operaciones de entrada y salida. String para el tratamiento de cadenas.






2Segundo paso

Se declaran las variables en "string" (primera y segunda) para los intentos se declaran en la variable int.






3Tercer paso

Se declara "void"; que declara el concepto de no exitencia a un tipo de variable.






4Cuarto paso

Se declara int main para declarar parámetros y se imprime texto con cout para dar información al jugador.






5Quinto paso

Se declaran las variables inicie y visible, para luego declarar while para dar inicio al ciclo del juego, esto junto a el contenedor char.






6Sexto paso

Se declara el condicional if para poder señalarle al jugador si gano.






7Septimo paso

Se declara el condicional else para poder señalar al jugador si ha perdido.






8Octavo paso

Se declara nuevamente void para llevar el conteo de los intentos restantes del jugador.






9Noveno paso

Se declara void junto con el numero de intentos y la palabra para adivinar, para luego declarar la funcionalidad del juego con ayuda del for, para ejecutarlo de forma repetitiva y con los condicionales if y else.






10Decimo paso

Se declara el ciclo para el conteo de los intentos con ayuda void, bool, for y el condicional if.
